Regra de Ouro da Base de Conhecimento

👉 O broker NUNCA deve ter que “fatiar” manualmente PDFs, slides ou relatórios.
O sistema é que deve fazer o trabalho sujo.

Upload único. Estrutura automática. Revisão só se quiser.

Fluxo Ideal para PDF / Slides Complexos (Comitê, Relatórios, Materiais Grandes)
1️⃣ Upload Simples (Experiência do Broker)

Tela: Upload Inteligente

O broker só faz:

Upload do PDF (ex: Comitê Janeiro.pdf)

Seleciona:

Tipo de material:

Comitê

Research

Produto

Campanha

Treinamento

Outro

(Opcional) Marca:

Período (Jan/2026)

Tags gerais (ex: renda fixa, fundos, estratégia)

Botão:
✅ “Processar automaticamente”

Fim. Zero recorte manual.

2️⃣ Pipeline Interno (Automático, invisível pro broker)

O sistema deve:

A. Quebrar semanticamente (não só por página)

Detectar:

Tabelas

Gráficos

Títulos

Seções

Slides focados em 1 produto

Slides comparativos

B. Criar Blocos de Conhecimento

Cada PDF vira múltiplos blocos internos:

Exemplos:

Bloco:
Tipo = Tabela Comparativa
Relacionado a = Produto A, Produto B, Produto C

Bloco:
Tipo = Deep dive Produto
Produto = Fundo XPTO
Fonte = Comitê Janeiro

Bloco:
Tipo = Gráfico
Classe = Fundos Multimercado
Período = Jan/2026

3️⃣ Auto-Relacionamento com Hierarquia

A IA deve tentar automaticamente:

Detectar nomes de:

Produtos

Fundos

Classes

Taxas

Estratégias

E linkar com:

Tipo → Classe → Ativo → Conteúdo

Mesmo que o broker não faça nada.

Lista de blocos criados:

Tabela comparativa (3 produtos)

Slide produto XPTO

Gráfico classe multimercado

Para cada bloco:

Produto(s) vinculados

Tipo detectado

Preview

O broker pode:

Ajustar produto/classe se quiser

Aprovar tudo em 1 clique

Mas se não mexer:
👉 Já entra na base automaticamente.

Como a IA vai “saber onde pesquisar”

A base deixa de ser só arquivos.

Ela vira:

Base = Blocos Semânticos + Relacionamentos

Então a IA pode:

Quando buscar por:

“taxa do fundo XPTO”

Ela consulta:

Blocos do tipo:

Tabela

Produto

Taxa

Relacionados ao:

Ativo = Fundo XPTO

Mesmo que isso esteja no meio de um PDF de 200 páginas.

Regra para o Replit (direta)

Regra de Produto para PDFs e Slides

Upload deve aceitar PDFs grandes

Sistema deve:

Extrair texto, tabelas e gráficos

Criar blocos internos semânticos

Auto-relacionar com produtos/classes

Broker NÃO deve:

Recortar páginas

Subir por produto

Busca da IA deve consultar:

Blocos internos, não arquivos
