Bug 1 — Checkbox do cabeçalho não seleciona todos os pendentes
A função toggleSelectAll() está correta internamente, mas o problema é de timing: ela chama renderDocuments() antes de updateSelectionBar(). Como renderDocuments() reconstrói o DOM inteiro, o selectAllCb é destruído e recriado, perdendo o estado visual. Funciona na lógica mas a UI não reflete. A correção é garantir que o estado dos checkboxes seja aplicado dentro do renderDocuments(), o que já acontece via selectedIds.has(Number(doc.id)) — mas o checkbox do cabeçalho precisa ser atualizado depois do render, não antes.
Bug 2 — Botão "Processar pendentes" clicável sem seleção
O botão do pendingAlert (linha 268) tem onclick="selectAllPending(); processSelected()" — ele seleciona tudo e já abre o modal imediatamente. O problema que você descreve ("só deve ficar ativo quando selecionar ao menos um") se aplica ao botão "Processar selecionados" da selectionBar. Esse botão não tem disabled aplicado dinamicamente — a função updateSelectionBar() só mostra/esconde a barra, mas não desabilita o botão quando selectedIds.size === 0. Como a barra só aparece quando há seleção, na prática não é um risco — mas se quiser travar o botão mesmo assim, é simples.
Bug 3 — Botão "Ver" não dispara nada
Linha 551: o botão usa onclick="window.open('/api/products/materials/${doc.id}/pdf', '_blank')" — isso está dentro de uma template string com crases, e o ${doc.id} é interpolado corretamente em JS. O problema real é que o onclick inline está dentro de uma string que é inserida via innerHTML. Quando o HTML é parseado pelo browser via innerHTML, event handlers inline em strings templateadas são bloqueados pela CSP (Content Security Policy) que a sua aplicação usa (nonce="{{ request.state.csp_nonce }}"). A CSP bloqueia onclick inline em conteúdo inserido dinamicamente sem nonce — por isso não aparece nada no console de rede.
A solução é usar event delegation no container, igual ao que já foi feito para os botões "Processar" e "Excluir" (linha 885-896). O botão "Ver" foi esquecido nessa refatoração.
